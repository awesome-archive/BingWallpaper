BingWallpaper

功能

  1. 下载指定天数 Bing 首页的图片
  2. 定时更新桌面壁纸
  3. 随系统自动启动

用法

  • BingWallpaper.exe /? : 查看命令行参数

  • BingWallpaper.exe --help : 查看命令行参数

  • BingWallpaper.exe -a : 开机启动

  • BingWallpaper.exe -i : 定时更新壁纸的间隔 （分钟为单位）

  • BingWallpaper.exe -r : 下载指定天数（N天） Bing 首页的图片，并在多张（Ｎ张）图片之间循环更新当前桌面壁纸
